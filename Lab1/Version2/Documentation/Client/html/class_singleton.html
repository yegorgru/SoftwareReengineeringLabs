Detailed Description

template<typename T>
class Singleton< T >

class implementation of Sinfleton pattern

There can't be more than one object of class in program Class T must be inheritant of Singleton<T> and have it as friend in order to inherit its singleton features NonMoveable, NonCopyable inheritant
